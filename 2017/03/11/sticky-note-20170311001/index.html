便利贴-小记hashchange

hashchange，监听浏览器地址锚点（”#”）后部分发生修改触发事件

利用此事件监听可实现简单的路由切换，根据”#”后部分的变化。在页面插入不同的html，实现类似于vue、react的路由。


你可以使用以下方式调用事件:
a.通过设置Location 对象 的 location.hash 或 location.href 属性修改锚部分。
b.使用不同书签导航到当前页面（使用”后退” 或”前进”按钮）
c.点击链接跳转到书签锚
